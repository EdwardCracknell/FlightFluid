Questions, answered

Does FlightFluid control my aircraft?

No. FlightFluid provides planning and guidance cues. Always maintain manual control and follow local regulations.

Will it work without external sensors?

Yes. Core features run on‑device. Wind‑aware guidance improves with supported data sources when available.

Can I export paths to my editor?

Pro users can export reference paths and stabilization data to common video editors and tools.

Is there a team plan?

Yes—Teams includes shared libraries, roles, and collaboration features for crews and organizations.
